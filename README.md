# AnyEnums
By AnyEnums, you can select multiple values on `enum`

comming soon

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'any_enums'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install any_enums
```
